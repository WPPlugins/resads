== Description ==

More and more users use mobile devices and see this ugly display errors.
Google has recognized this and is mobile optimized websites better positions in the search results.
Here, the advertising media are often overlooked. Remedy this creates ResAds.
ResAds provides, depending on the user's terminal, the optimum display size is achieved with just a few clicks.

== Installation ==

1. Upload `resads` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the menu AdManagement and create one ad
4. Go to the menu AdSpots and create one AdSpot
5. Copy created Shortcode and put them into an post or create an widget

= German =

1. Lade `resads` in den Wordpress Plugin Ordner `/wp-content/plugins/`
2. Aktiviere `resads` im Plugin Menu von Wordpress
3. Gehe zum Menüpunkt Werbeverwaltung und erstelle eine Anzeige
4. Gehe zum Menüpunkt AdSpots und erstelle ein AdSpot
5. Kopiere den erstellten Shortcode und füge ihn in ein Artikel ein oder erstelle ein Widget

== Frequently Asked Questions ==

= Can all ads be tracked =

No. Iframes and some JavaScript Codes cant be tracked.

= Why is my link not masked, although I selected it? =

In order to use the options "link mask", "set link to nofollow" and "open link in new window" you can not use the "source code".

= How can i create an Ad? =

1. Go to menu AdManagement and click "Add new Ad"
2. Carrying a banner name
3. Carrying a target url and banner url OR a source code and banner size - You don't need both! -
4. Select options
5.Select AdSpot(s)
6. Submit

= Deutsch =

= Können alle Anzeigen getrackt werden? =

Nein. Iframes und einige JavaScirpt Codes können derzeit nicht getrackt werden.

= Warum ist mein Link nicht maskiert, obwohl ich es ausgewählt habe? =

Um die Optionen "Link maskieren", "Link auf nofollow setzen" und "Link im neuem Fenster öffnen" nutzen zu können, darfst Du den Quellcode nicht einfach so einfügen.

= Wie erstelle ich einen Banner? =

1. Gehe zum Menüpunkt "Bannerverwaltung" und klicke "Neuer Banner" an
2. Trage einen Bannernamen ein
3. Trage die Ziel-URL und Banner-URL ODER den Quellcode und die Bannergröße ein
4. Du brauchst nicht beides eintragen!
5. Wähle die Optionen
6. Wähle die AdSpot(s)
7. Absenden
